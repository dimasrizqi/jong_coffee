## Installation

1. `git clone "https://github.com/tanweikang02/SDP.git"`
2. `git remote add origin "https://github.com/tanweikang02/SDP/main"`
3. `git branch --set-upstream-to=origin/<branch>`
4. `git pull`

### Useful Git Commands

- `git add`
- `git commit`
- `git push`
- `git stash`
